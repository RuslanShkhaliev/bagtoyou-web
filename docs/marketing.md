# Problem

Existing marketplaces are:

- Overloaded with features
- Slow to use on mobile
- Require complex onboarding
- Push paid visibility aggressively

> Users struggle to quickly sell or find items without friction.

---

# Solution

This product is a simple, fast, and honest flea market:

- Minimal interface
- Mobile-first experience
- Listing creation in 1–2 minutes
- No visual or UX noise

---

# Target Audience

- Individuals aged 18–45
- Occasional sellers and buyers
- Mobile-first users
- People who value speed and simplicity

---

# Value Proposition

- **Fast listing creation**
- **Clear navigation**
- **Low cognitive load**
- **Trust through simplicity**

---

# Core Use Cases

1. Sell unused items
2. Discover nearby offers
3. Quickly contact sellers

---

# Positioning

> A marketplace that respects user time and removes unnecessary complexity.

---

# Future Expansion

- Local discovery
- Smarter categorization
- Personal recommendations

_Without compromising simplicity._
